Public method define

Sets a cookie given a value of any type.

Parameters:

  • name must be a String.
    (The name of the cookie to be set)

  • value can be of any type.
    (The value to convert to string and set in the cookie)

  • options is optional and must be an Object.
    (Options hash)

    • options.expires is optional and can be of any type.
      (The expiration as a number of seconds, or "session", or undefined for one year)

Returns a Boolean
(Whether or not the cookie was successfully set)

Example:

cookie.set('foo', 'bar', { expires : 1000000 });
cookie.set('foo', [1, 2, 3], { expires : 'session' });
cookie.set('foo', { bar : 'baz', boom : 'boosh' });
